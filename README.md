# 📖 Dictionary Application

A web-based Dictionary Application developed using **Node.js, Express.js, SQLite, HTML, CSS, and JavaScript**. The application allows users to search words, add new words, update meanings, delete words, import multiple words, and receive search suggestions.

---

## 🚀 Features

- 🔍 Search words with meanings
- ➕ Add new words
- ✏️ Update existing words
- 🗑️ Delete words
- 📥 Import words from a text file
- 💡 Search suggestions (Autocomplete)
- ⭐ Favorite words
- 🕒 Recent searches
- 🌙 Dark Mode
- 📊 Total word count
- 💾 SQLite database for data storage

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Express.js
- SQLite3

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/Suhani-kumari-007/dictionary-application.git
```

### 2. Open the project

```bash
cd dictionary-application
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the server

```bash
node server.js
```

### 5. Open in browser

```
http://localhost:3000
```

---

## 📁 Import File Format

The application imports words from a text file in the following format:

```text
apple
A sweet fruit

cat
A domestic animal

dog
A loyal animal
```
